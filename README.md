# YAMA

Yet Another Movies App

---

## Setup

Add a `.env` file in the source folder, with the following parameters:

```
REACT_APP_MOVIE_DB_KEY=string
REACT_APP_MOVIE_DB_PATH=string
REACT_APP_MOVIE_DB_IMAGE_PATH=string
```
